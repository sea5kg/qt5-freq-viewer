# qt5-freq-viewer


## Ubuntu


Requirements for build:
```
$ sudo apt install qtdeclarative5-dev qtquickcontrols2-5-dev
```


## Run from command line:

Run with filepath with data:
```
$ ./qt5-freq-viewer testdata/test1.txt
```
